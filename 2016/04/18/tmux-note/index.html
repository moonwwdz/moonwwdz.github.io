    tmux使用及快捷键

    2016-04-18
    linux, tmux, 备忘

    工作区域的一些概念区分

    tmux中的工作区域，就个人的理解来看，可以分为三种，会话,窗口,面板。可以使用日常使用的excel来说明一下。

    一个会话相当于新打开的一个excel文件，是完全独立的一个工作区域;
    一个窗口相当于excel中的sheet，是属于一个会话中的工作区域;
    而面板，就是你在一个sheet中为标记不同的工作内容而随手划拉的一个部分;

    会话的使用

    • tmux new -s 'you work place title'新开启一个会话，可以用来区分不同的工作
    • tmux ls 或者 ctrl-b s 列出所有的会话
    • tmux a 接入第一个可用的会话，第一列是会话的标识
    • tmux a -t '<you work place title >'接入一个选定的会话
    • ctrl+b d离开会话
    • tmux kill-session -t session-name 关闭会话
    • ctrl+b $重命名会话

    窗口的使用

    • ctrl+b c 新建一个窗口，光标自动移到新的窗口，在下方的标题栏中会多出一个窗口的标识(有数字可以区分)
    • ctrl+b <num> 快速移到到这个窗口（num是第几个窗口）
    • ctrl+b & 关闭一个窗口
    • ctrl+b , 重命名当前窗口
    • ctrl+b w 列出所有窗口
    • ctrl+b % 水平分割窗口
    • ctrl+b " 竖直分割窗口
    • ctrl+b n 选择下一个窗口
    • ctrl+b p 选择上一个窗口

    分割面板的使用

    • ctrl+b % 创建一个水平窗格
    • ctrl+b " 创建一个竖直窗格
    • ctrl+b h 将光标移入左侧的窗格
    • ctrl+b j 将光标移入下方的窗格
    • ctrl+b l 将光标移入右侧的窗格
    • ctrl+b k 将光标移入上方的窗格
    • ctrl+b q 显示窗格的编号
    • ctrl+b o 在窗格间切换
    • ctrl+b } 与下一个窗格交换位置
    • ctrl+b { 与上一个窗格交换位置
    • ctrl+b ! 在新窗口中显示当前窗格
    • ctrl+b x 关闭当前窗格
