    2018-04-03
    linux

    Emacs 中一些简单的配置,不需要太折腾就可以很好的使用

    快捷键

    Alt快捷键

    按键 作用
    M-; 插入目前語言的註解
    M-! 执行命令
    M-g M-g 跳到第 N 行
    M-Page Down 隔壁的 window 向下捲動一頁
    M-Page Up 隔壁的 window 向上捲動一頁
    M-% 字串搜尋並取代
    M-l 單字轉成全小寫
    M-u 單字轉成全大寫
    M-c 單字轉首字大寫
    M-t 前後單字交換

    M-% 替换的词会高亮起来，这时，输入y替换并跳到下一个，输入n忽略并跳到下一个，输入q结束，输入！替换剩下的全部

    Ctrl-x 快捷键

    按键 作用
    C-x C-w 另存新檔
    C-x k 關掉目前 buffer
    C-x C-← 切換到上一個 buffer
    C-x C-→ 切換到下一個 buffer
    C-t 前後字元交換
    C-x C-e 把游標移動到括弧最尾端後設定就立即生效
    C-c C-e h 导出org为html
